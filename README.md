## Project Workflow

### 1. Data Loading

* Imported dataset into Python environment.
* Performed initial inspection of data structure and data types.

### 2. Exploratory Data Analysis (EDA)

* Examined data distribution and patterns.
* Identified trends, correlations, and anomalies.
* Generated visualizations to understand key metrics.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Corrected inconsistent data formats.
* Prepared clean and analysis-ready datasets.

### 4. SQL Analysis

* Loaded cleaned data into MySQL.
* Wrote SQL queries to:

  * Aggregate key metrics
  * Perform filtering and segmentation
  * Generate business insights
  * Validate analytical findings

### 5. Dashboard Development

* Connected Power BI to processed data.
* Created interactive dashboards and visual reports.
* Developed KPI tracking and trend analysis views.

### 6. Reporting & Presentation

* Documented insights and recommendations.
* Prepared business report summarizing findings.
* Created presentation slides using Gamma and PowerPoint.

## How to Run

### Prerequisites

* Python 3.x
* MySQL Server
* Power BI Desktop

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/project-name.git
```

2. Navigate to the project folder:

```bash
cd project-name
```

3. Install required Python packages:

```bash
pip install -r requirements.txt
```

4. Run the Jupyter Notebook for EDA and data cleaning.

5. Import the cleaned dataset into MySQL and execute SQL scripts.

6. Open the Power BI file (`.pbix`) to explore the dashboard.
